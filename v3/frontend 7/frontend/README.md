# GharBanao — India's AI-Powered Construction Operating System

A world-class marketing homepage for **GharBanao**, built with React + Vite + Tailwind CSS
and Framer Motion. Matches the approved hero design and implements the full 12-section vision.

## Stack
- **React 18** + **Vite 5**
- **Tailwind CSS 3** (custom forest/cream/gold/neon design system)
- **Framer Motion** (calm, intelligent animations)
- **lucide-react** (icons)

## Run locally
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## Sections
1. Hero — headline, badges, CTAs, live dashboard cards, AI assistant search bar
2. One Platform — six audience cards
3. The Construction Ecosystem — interconnected AI engine nodes
4. Live Project Intelligence — mission-control dashboard
5. Planning → Possession band + 9-step homeowner journey
6. AI Permit Navigator — plan compliance check + expert connect + tracking
7. Civil Engineering Hub — careers, tools, equipment rentals
8. Powerful Features — phone mockup, feature grid, video card
9. Trust Engine — verification layers
10. AI Site Monitoring — drone/photo analysis overlays
11. Live Material Market — price board + ticker
12. Escrow & Finance — milestone vault + financing
13. Stats + Trust badges
14. Future of Construction — cinematic statement
15. Final CTA + Footer

## Design system
- **Colors:** deep forest green, rich cream, matte gold, graphite; neon green (AI) + warm orange (construction) accents
- **Type:** Plus Jakarta Sans (display), Inter (body), Fraunces (serif statements)
- **Motion:** slow floating, data-pulse, scroll reveals — "calm intelligence", not startup chaos
